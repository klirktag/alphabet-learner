## The technology
This should be implemented with web technology

Preferably raw html, javascript, css and jquery

Avoid complicated frameworks like React

## The app
We want to make a simple app for kids for learning the alphabet

Make a keyboard the size of the screen (or web view in the browser)

The keyboard should be in alhabetic order and not querty because it is for pedagogic purpose

The keyboard should be colorful and children friendly themed

When a key on the keyboard is pressed, a sound should be played for that letter

Make a folder "sound", and create recordings of each audio in this folder. You can use whatever command line tools or other tools you see fit


## Workflow
After each change make a commit and start the web application for trying it out
